# Install PayPal SDKs

Run this command to install the official PayPal Node.js SDK:

```bash
npm install @paypal/checkout-server-sdk
```

Or if using yarn:

```bash
yarn add @paypal/checkout-server-sdk
```

This SDK provides:
- ✅ Official PayPal server-side SDK
- ✅ Type definitions
- ✅ Built-in error handling
- ✅ OAuth token management
- ✅ Request/response models

The JavaScript SDK will be loaded dynamically via CDN (no npm package needed).
